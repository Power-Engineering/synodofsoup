🛠 Setup Guide

  1. Go to supabase.com and create a free account
  2. Create a new project (takes ~2 minutes to provision)
  3. In your project, go to SQL Editor and run the SQL in schema.sql
  4. Go to Project Settings → API and copy your Project URL and anon/public key
  5. Open config.js and paste those two values in
  6. Done — reload the page and comments will be live!
